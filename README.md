# NutriOne - Personal AI Fitness & Nutrition

O **NutriOne** é uma plataforma integrada de saúde que utiliza Inteligência Artificial para democratizar o acesso a planos de nutrição e treino personalizados. Mais do que um projeto pessoal, este foi um **MVP lançado no mercado** para validar a adesão de utilizadores reais a soluções de bem-estar assistidas por IA.

## Funcionalidades Inovadoras

* **AI Nutritionist & Coach:** Geração de planos de dieta e treino adaptativos com base no perfil biométrico e objetivos do utilizador.
* **Smart Food Scanner:** Integração de Computer Vision para identificar alimentos, calorias e macronutrientes a partir de uma simples fotografia.
* **Workout Tracker & Gallery:** Biblioteca de exercícios com registo de carga e histórico de progresso (Log de treinos).
* **Interactive Dashboards:** Visualização clara da evolução do peso, consumo calórico e performance física.
* **User Profile & Cloud Sync:** Gestão de conta e persistência de dados em tempo real.

## Stack Tecnológica

* **Frontend:** React + TypeScript (Vite) para uma interface rápida e fluida.
* **Estilização:** Tailwind CSS e Shadcn/ui para um design moderno e responsivo.
* **Backend & Database:** Supabase (PostgreSQL, Auth e Storage).
* **Infraestrutura:** Cloud-native, focado em escalabilidade para o lançamento do MVP.
* **IA:** Integração de modelos de linguagem e visão computacional.

## O Desafio do MVP
Este projeto foi desenvolvido com foco em **agilidade e time-to-market**. Utilize a plataforma Lovable para acelerar o desenvolvimento do frontend e infraestrutura, permitindo-me focar na lógica de integração da IA e na experiência do utilizador final.
